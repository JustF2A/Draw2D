# Svg.Skia

SkiaSharp SVG Rendering Library

## Usage

```
Usage: Svg.Skia.Converter -f path [-o path]
       Svg.Skia.Converter -d path [-o path]
```

## Externals

The `Svg.Skia` library is using code from the https://github.com/vvvv/SVG
